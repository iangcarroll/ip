# ip.ian.sh

A minimal "what is my IP" tool. Shows your IPv4 and IPv6 addresses with geo/ASN info via [ipinfo.io](https://ipinfo.io).

```
$ curl ip.ian.sh
203.0.113.42

$ curl ip.ian.sh/json
{"ip":"203.0.113.42","version":4,"city":"San Francisco","region":"California",...}
```

## How it works

- `ip.ian.sh` — A + AAAA records, connects via whichever protocol wins
- `ip4.ian.sh` — A record only, forces IPv4
- `ip6.ian.sh` — AAAA record only, forces IPv6

The frontend fetches the alternate protocol IP via the explicit subdomains.
